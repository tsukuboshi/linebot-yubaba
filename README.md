# Linebot-Yubaba

&emsp;
## Overview

This repository contains codes to deploy yubaba line bot.

&emsp;

## Required

- Channel (Line Messaging API)
- AWS CLI
- SAM CLI
- Docker(option)

&emsp;

## Parameters (Line Messaging API)
- Use webhook : Enabled
- Auto-reply messages : Disabled
- Greeting messages : Enabled
